# 村山良作 無印方向與外殼整理

更新日期：2026-03-12

## 1. 現在的判斷

### A. 頁面本身

目前切出來的 `村山良作` 頁面已經比舊站清楚很多，但還偏向：

- 設計稿感
- proposal / works site 感
- 工業感比較重

如果要再往你說的 `無印良品感` 靠，方向應該不是變成更文青，而是：

- 更安靜
- 更克制
- 更少裝飾性對比
- 更像一個有秩序的品牌頁

### B. WordPress 外殼

如果沿用現在舊站的全站：

- top menu
- footer
- 漂浮按鈕
- 預設頁寬
- 可能的 sidebar / breadcrumb / 分享元件

那很高機率會破壞現在這批頁面的感覺。

結論：

`頁面本身要改，外殼也要一起整理。`

## 2. 什麼叫「更有無印感」

不是去模仿 MUJI 品牌，而是借它的幾個核心特質：

1. 留白更大
2. 版面更平靜
3. 線條更少但更準
4. 色彩更接近紙、灰、木、布
5. CTA 更低調，但仍然清楚

對 `村山良作` 來說，具體會變成：

- 背景更接近米白、紙灰、淺木色
- 深色區塊比例減少
- 黑白對比不要那麼硬
- 卡片邊框更淡
- 標題與內文之間的節奏再放鬆一點
- 按鈕更像靜態標籤，不像很強勢的 SaaS 按鈕

## 3. WordPress 外殼一定要一起整理的項目

### A. Top Menu

如果舊站 top menu 還是沿用原本 WordPress 主題樣式，會直接讓新頁面破功。

至少要整理：

- logo 尺寸
- menu 字級
- 行高與高度
- 背景色
- hover 樣式
- 是否 sticky

### B. Footer

如果 footer 還是舊站那種資訊很多、很雜的型態，也會把整體拉回舊站感。

MVP 建議 footer 先極簡：

- 品牌名
- 聯絡方式
- IG / Line
- 版權

不要先塞太多連結。

### C. 漂浮按鈕

這個最容易破壞版面。

如果現在站上的漂浮按鈕：

- 顏色很跳
- 陰影很重
- 尺寸太大
- 位置固定在畫面右下

那它會直接把頁面的安靜感打掉。

建議：

- `村山良作` 頁面先停用舊漂浮按鈕
- 改成同風格的簡化版 CTA
- 或只保留一個低干擾的聯絡按鈕

### D. 頁寬與頁面模板

如果 WordPress 主題預設不是 full-width，或會自動包一層 container，版面也會被擠壞。

MVP 最穩做法：

- 為 `村山良作` 做一個專用 page template
- 使用 full-width
- 關掉 sidebar
- 關掉分享元件
- 關掉預設文章 meta 區塊

## 4. Logo 看完後的判斷

### 村山

`logo/村山/組合04.png`

這個 logo 很有個性，偏：

- 工具感
- 手作感
- 黑白強烈
- 品牌主體性很高

所以 `村山良作` 的頁面不需要再很花。
反而應該讓版面更安靜，讓 logo 自己說話。

### 村花

`logo/村花/漢字村花.png`

這個 logo 本身就比 `村山` 更戲劇化，也更有婚禮感。
所以 `村花弄囍` 可以保留比較細膩和柔美的視覺。

### 鮭女

`logo/鮭女/魚頭LOGO.png`

這個 logo 很強，畫面感重，也帶有商品與品牌識別潛力。
鮭女更適合做成品牌電商感，而不是完全極簡。

## 5. 對村山良作最實際的做法

如果目標是：

`先讓村山良作成為第一個可上線的新頁面`

那我建議分兩層做：

### 第一層：頁面本體

先把這三頁做成更安靜的版本：

- 首頁
- 案例總覽頁
- 合作流程頁

### 第二層：外殼

再補一份 `村山良作專用外殼`：

- top menu
- footer
- CTA 按鈕
- 頁面寬度規則

## 6. MVP 最佳實作方式

不是把整個 WordPress 主題全部重做。

而是先做：

1. `村山良作` 專用頁面模板
2. `村山良作` 專用 header / footer 樣式
3. 停用舊的漂浮按鈕或改成低干擾版本

這樣成本最低，效果最好。

## 7. 下一步

最合理的順序是：

1. 先把 `村山良作` 三頁再收成更安靜的無印方向
2. 再補 `村山良作專用外殼` 原型
3. 最後再整理成 WordPress 落地版本
