## 2.1 首頁可替換案例

### 暖暖戶政變身

來源：
- `https://www.weddingwishlove.com/2024/08/01/blog_20240801/`

適合放的位置：
- 首頁精選案例卡
- 案例總覽頁 `展場與空間 / 拍照區` 類型

首頁短標：
- 暖暖戶政變身

首頁一句話：
- 把原本普通的登記空間，整理成新人會想停下來拍照的場景。

案例卡摘要：
- 這類案例的價值不只是「改造前後差很多」，而是把原本沒有記憶點的空間，轉成能承接情緒、拍照與現場體驗的節點。

標籤：
- 空間改造
- 拍照區
- 戶政場景

首頁用圖建議：
- 首頁卡片只用 `After` 圖
- 不用 `Before` 圖當首頁首圖或首頁精選第一屏
- 如果要帶出改造力，優先用 `After` 圖 + 一句「前後整理」文案
- `Before / After` 對比保留到單篇作品頁內再完整呈現

原因：
- `Before` 圖敘事很強，但視覺噪音也高
- 放在首頁第一層容易破壞 `村山良作` 目前的安靜、克制方向
- 首頁應該先建立質感，再讓人進案例裡看改造幅度

## 4. 首頁區塊直接可用文案

### 品牌介紹短版

- 村山良作處理的不是單一佈置物，而是品牌活動、主題場景與空間現場的整體感受。
- 從展場、品牌活動到開幕現場，我們在意的是每一個人走進來時的第一眼感覺。
- 讓空間不只被看見，也能替品牌說話。

### Hero 定位句版本 B

- 把品牌活動、主題場景與空間現場，整理成真的可被感受的體驗。

### Hero 定位句

- 把品牌活動、主題場景與展場空間，做成真的現場。

### Hero 補充說明

- 從展場、品牌活動到開幕現場，村山良作關注的不只是佈置，而是每一個「第一眼感覺」的塑造。
- 我們把空間、動線、拍照點與現場氣氛整理成同一種語言，讓活動不只是完成，而是真的成立。

### 服務帶短句

- 品牌活動：把品牌訊息做成可停留、可拍攝、可被記住的活動現場。
- 主題場景：把世界觀做成立體的空間，而不是平面的佈置。
- 展場與空間：把主題、導視與停留節奏整合成可被理解的現場。
- 現場執行：把設計方向真正落到活動當天的條件裡。

### 案例入口標題

- 先看哪一類案例最接近你的需求。

### 精選案例區導言

- 有些案例重點在品牌活動的現場感，有些在主題世界的完整度，也有些是在既有空間裡做出前後差異。先從最接近你需求的那一類開始看。

### CTA 文案

- 如果你已經看到接近的案例，下一步就不是再繼續滑，而是把活動類型、場地、日期和預算區間交給我們。

## 5. WordPress 上線前最低補件

每個案例至少補：

1. 封面圖 1 張
2. 文章連結
3. 最終標題
4. 最終一句摘要

如果首頁要改用 `暖暖戶政變身`，另外補：

5. `After` 圖 1 張作為首頁精選案例封面
6. `Before / After` 對比保留在單篇作品頁，不直接搬到首頁主視覺
