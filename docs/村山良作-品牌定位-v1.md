# goodjob 定位整理 V1

更新日期：2026-03-11

## 1. 目前判斷

`goodjob / 村山良作` 不適合再被介紹成很空泛的「商業活動」。

根據目前官網公開內容，它更適合被定義成：

`主題場景與品牌活動工作室`

這個定義比「商業佈置」更完整，因為它同時包含：

- 品牌活動
- 展場與拍照區
- 主題派對
- 公部門與企業活動現場
- 粉絲應援與特殊主題現場

## 2. 現有內容透露出的真實能力

從目前公開案例看，goodjob 已經不是單一類型服務，而是有幾條很明顯的線：

- 展覽 / 公共空間
  - 成美文化園昆蟲展佈置
  - 大安戶政 / 大同戶政拍照區

- 品牌 / 商業活動
  - 2024 華歌爾週年慶活動佈置
  - Lativ 魔法月台派對佈置

- 社群 / 粉絲 / 主題活動
  - G-Dragon 粉絲應援佈置
  - 戀與深空花藝佈置
  - 特務舞會派對

- 課程 / 社群型活動
  - 曾蘋果老師課程活動佈置
  - CXO 女創俱樂部拍照區佈置

所以它最該強調的不是「我們也能做非婚禮」，而是：

`我們把主題、品牌與現場做成可被體驗、可被拍照、可被記住的空間。`

## 3. 品牌角色

如果三品牌一起看：

- `Weddingwish`
  - 婚禮場景工作室
  - 方案型錄 + 客製婚禮

- `salmonbelle`
  - 花藝選物品牌
  - 花藝商品 + 陪伴型 IP

- `goodjob`
  - 主題場景與品牌活動工作室
  - 現場體驗 + 商業合作 + 空間執行

這樣三牌分工才會清楚。

## 4. goodjob 不該怎麼介紹

不建議用：

- 商業活動
- 非婚禮活動作品集
- 商業佈置品牌

原因：

- 太泛
- 太被動
- 看不出主題能力
- 看不出品牌合作與現場執行能力

## 5. 建議對外說法

### 長版

`村山良作，專注把品牌活動、主題派對、展場與拍照區，做成可被感受、可被記住的現場。`

### 中版

`主題場景與品牌活動工作室`

### 短版

`隱於村山，現於良作`

## 6. 首頁語氣方向

goodjob 首頁不該像婚禮品牌那樣柔軟，也不該像花店那樣感性。

它比較適合：

- 提案式
- 案例式
- 清楚分服務
- 直接回答合作方會問的問題

首頁關鍵不是「說很多美感」，而是先讓人理解：

- 你們做什麼型態的專案
- 有哪些合作類型
- 執行方式是什麼
- 看哪些案例最 relevant

## 7. 站內應該強調的四條能力線

### A. 品牌活動

- 週年慶
- 品牌派對
- 快閃活動
- 宣傳活動

### B. 主題場景

- 派對主題
- 粉絲應援
- 世界觀型現場
- 沉浸式拍照區

### C. 展場 / 公共空間

- 展覽空間
- 拍照區
- 公部門活動現場

### D. 現場執行整合

- 視覺設定
- 裝置安排
- 現場搭建
- 成果落地

## 8. 一句話版本

goodjob 的核心不是「做非婚禮」，而是：

`把品牌、主題與空間，做成能被體驗的現場。`
