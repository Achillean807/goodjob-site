# 村山良作 外殼規格 V1

更新日期：2026-03-12

## 1. 適用範圍

這份規格只對應 `村山良作` 第一階段 MVP。

對應頁面：

- `prototypes/村山良作-首頁-MUJI版.html`
- `prototypes/村山良作-案例總覽頁-MUJI版.html`
- `prototypes/村山良作-合作流程頁-MUJI版.html`

## 2. Header

### 目標

- 保持安靜
- 讓 logo 有位置，但不要過度搶畫面
- 不帶舊站主題感

### 結構

- 左：logo + goodjob / 村山良作
- 中：主選單
- 右：簡化 CTA

### 主選單

- 服務範圍
- 案例入口
- 洽詢合作

內頁版：

- 首頁
- 案例總覽
- 合作流程

## 3. Footer

### 目標

- 資訊少
- 不雜
- 不像舊站 footer

### 第一階段只放

- 頁面名稱
- 品牌識別字

如果要補聯絡資訊，也建議極少量：

- Line
- IG
- Email

## 4. 漂浮 CTA

### 原則

- 保留一個就好
- 低干擾
- 中性色
- 不用鮮豔色塊

### 文字建議

- 合作洽詢

### 不建議

- 多顆浮動按鈕
- 亮綠色 Line 圓鈕
- 很大的陰影

## 5. Page Template

WordPress 建議：

- full-width
- 關掉 sidebar
- 關掉文章 meta 區塊
- 關掉分享元件
- 關掉預設 breadcrumb

## 6. 視覺方向

### 關鍵字

- 安靜
- 紙感
- 留白
- 低對比
- 有秩序

### 不要的方向

- 舊站感
- 很重的商業按鈕感
- 太多深色區塊
- 過度強調 hover / 動畫

## 7. Logo 使用

目前較適合這條線的是：

- `logo/村山/組合12.png`

理由：

- 比彩色版更安靜
- 比純黑大圖更容易融入頁面
- 保留村山良作的工具感和手作感

## 8. 實作建議

如果先做 MVP，不用整個 WordPress 主題重做。

先做：

1. 村山良作專用 page template
2. 專用 header / footer 樣式
3. 專用浮動 CTA

這樣就足夠把官網裡的 `村山良作` 入口換成新頁。
